# Contributing

Clone this repository

```bash
git clone https://github.com/tem2021/csc3002_3d_raycaster_game.git
```

Please do not commit to the main branch, the code may need further edits. 

Instead, [create a new branch](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-and-deleting-branches-within-your-repository). You can either name it the new feature you are adding (e.g. rewriteToCppStyle) or your name (e.g. Praemai).

[Make a pull request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request#creating-the-pull-request) to the main branch. Other team members can test your changes, it will be merged to the main branch after discussion and possible edits.
